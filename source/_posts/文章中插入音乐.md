实现效果:在一些需要的文章或页面中插入想要播放的音乐,让网站多一点点乐趣

<!-- more -->

在本地hexo目录中安装插件`npm install hexo-tag-aplayer` `npm install aplayer --save`

### 可以绕过版权的方法

#### 实例(**单首**)

```

```
