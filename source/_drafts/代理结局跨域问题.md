# 一级标题

```javascript
  proxyTable: {
    '/api': {
        target: 'http://localhost:8888'，
        changeOrigin: true，
        pathRewrite: {
            '^/api': ''
        }
      }
    }
```
`/api` 匹配所有带有`/api`的地址
`target` 后台地址
`changeOrigin` 跨域需要开这个
`pathRewrite` 省略`/api`发出请求
